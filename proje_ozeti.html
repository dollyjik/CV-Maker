1. Genel Yapı

Tamamen statik, backend'siz, 3 dosyadan oluşan bir web uygulaması. Node.js, sunucu veya veritabanı gerektirmez — tarayıcıda açmak yeterlidir.

DosyaSatırGörev
index.html438UI iskelet, form alanları, navigasyon
app.js1226Tüm mantık: state, CRUD, önizleme, PDF
styles.css366Temalar, layout, CV görünümü

Dış bağımlılıklar:

  • html2pdf.js v0.10.1 — CDN, PDF export için (html2canvas + jsPDF bundle)
  • Font Awesome 6.5.0 — CDN, ikonlar

2. Mimari & State Yönetimi

Herhangi bir framework kullanılmamış; vanilla JS ile reaktif önizleme sağlanmaktadır. Her input değişikliğinde updatePreview() → buildCVHTML() → DOM güncellenir.

Global değişkenler

let currentTheme = 'classic'   // seçili tema adı
let currentCVId = null          // localStorage'daki CV anahtarı
let photoDataUrl = null         // base64 fotoğraf string
const STORAGE_KEY = 'cvmaker_cvs'

Veri akışı

Form → collectData() → JSON objesi → buildCVHTML() → HTML string → cvPreview.innerHTML

Kayıt: collectData() → JSON objesi → localStorage.setItem()

Yükleme: localStorage.getItem() → JSON parse → restoreList() / restoreSkills() → DOM'a yazılır

PDF export konfigürasyonu

const opt = {
  margin: 0,
  filename: name + '_CV.pdf',
  image: { type: 'jpeg', quality: 0.98 },
  html2canvas: {
    scale: 2,           // yüksek kalite
    useCORS: true,
    logging: false,
    backgroundColor: '#ffffff'
    // windowWidth kasıtlı olarak KALDIRILDI — responsive CSS breakpoint sorunu
  },
  jsPDF: { unit: 'mm', format: 'a4', orientation: 'portrait', compress: true },
  pagebreak: { mode: ['css', 'legacy'] }  // çok sayfalı destek
};

PDF sayfalama için CSS kuralları: page-break-inside: avoid → .cv-section, .cv-item, .ep-timeline-entry, .eu-lang-table

7. Save / Load / Delete Sistemi

Kaydetme (saveCV())

  • Zorunlu alan: firstName veya lastName veya title
  • Her CV cv_{timestamp} ID'si alır
  • QuotaExceededError yakalanırsa fotoğrafsız tekrar dener (~5MB localStorage limiti)

Yükleme (loadCV())

  • resetForm() ile form temizlenir
  • restoreList() ile dinamik öğeler (deneyim, eğitim, dil vb.) yaratılır
  • restoreSkills() ile beceri satırları restore edilir

JSON Yedekleme & Geri Yükleme

9. Çözülen Kritik Sorunlar

SorunKök NedenÇözüm
PDF'de ekstra boş sayfa min-height: 1123px + cv-body min-height toplamı 1 sayfayı aşıyordu CSS gradient sidebar yaklaşımı; sidebar div'inin min-height kaldırıldı
PDF sadece sağ yarıyı gösteriyordu windowWidth: 794 parametresi responsive CSS'i tetikliyor, preview paneli daralıyordu windowWidth/windowHeight parametreleri kaldırıldı
PDF export tamamen bozuldu window.jsPDF / window.html2canvas bundle'dan ayrı erişilemiyor html2pdf().set().from().save() zinciri kullanıldı
Kaydet butonu çalışmıyordu localStorage.setItem() try-catch olmadan fırlatıyordu (QuotaExceededError) try-catch + fotoğrafsız fallback eklendi
Bölümler kaydediliyor ama yüklenmiyor (içerik boş) querySelectorAll('[id^="type_"]') hem container hem child input'ları eşleştiriyordu; nodes[nodes.length-1] son input'u döndürüyordu Selector .dynamic-item[id^="type_"] / .skill-item[id^="cat_skill_"] olarak güncellendi

10. Bilinen Sınırlamalar

  • Fotoğraf base64 olarak localStorage'da saklanır → büyük fotoğraflarda ~5MB limiti aşılabilir (QuotaExceededError handling mevcuttur)
  • PDF export masaüstü tarayıcılarda çalışır; mobilde responsive CSS sorunları yaşanabilir
  • Tüm veriler tarayıcı localStorage'ında tutulur — farklı cihaz/tarayıcıya geçişte JSON yedek/geri yükleme gerekir
  • Veri arama, filtreleme veya sıralama özelliği mevcut değildir
